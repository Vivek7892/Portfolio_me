# Contact Form Setup with Web3Forms

## Quick Setup (5 minutes)

### Step 1: Get Your Access Key
1. Go to [Web3Forms](https://web3forms.com)
2. Enter your email: **vivekvvivekv70@gmail.com**
3. Click "Create Access Key"
4. Copy the access key you receive

### Step 2: Add Access Key to Your Website
1. Open `js/script.js`
2. Find line with: `formData.append('access_key', 'YOUR_WEB3FORMS_ACCESS_KEY');`
3. Replace `YOUR_WEB3FORMS_ACCESS_KEY` with your actual key
4. Save the file

### Step 3: Test Your Form
1. Open your website
2. Fill out the contact form
3. Submit it
4. Check your email (vivekvvivekv70@gmail.com) for the message

## Alternative: Formspree (Backup Option)

If you prefer Formspree:
1. Go to [Formspree](https://formspree.io)
2. Sign up with vivekvvivekv70@gmail.com
3. Create a new form
4. Update the form in `index.html`:
   ```html
   <form action="https://formspree.io/f/YOUR_FORM_ID" method="POST" class="contact-form" id="contactForm">
   ```
5. Remove the JavaScript fetch code in `script.js`
